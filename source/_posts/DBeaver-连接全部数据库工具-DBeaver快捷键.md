Windows、Linux、Mac OS X多个平台都可以用

据我观察这个数据库可视化工具很不错，基于java

以各种驱动来连接数据库，也就是说java支持的数据库都可以用他连接

挺好用的，免费！！！

安装之后新建连接，选择你要链接的数据库，配置一下就好了。

快捷键
---
```
#Shift + Home选中当前光标到行首
#Shift + End选中当前光标到行尾
#Shift + ↑/↓/←/→ 移动光标并且选中
#Alt + X 执行选中的sql
#Ctrl + Enter 执行当前光标所在行的sql
#Ctrl + Alt + F 格式化SQL（file -> properties -> +SQL Editor -> SQL formatting）
```
