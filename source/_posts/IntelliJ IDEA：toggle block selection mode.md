在eclipse中有列编辑模式：toggle block selection mode

在idea中也可以，而且还比较高级，哈哈

idea -> 右上角 -> Edit -> Column Selection Mode -> 移动光标到你想要弄的行

完事在重复一次，就可以退出列编辑模式
